# Evidence Consistency Skill

Keep the narrative aligned with the artifacts. `interpretation.md` must not
claim a discovery, clear excess, successful fit, or strong conclusion unless the
numeric artifacts and trace support it.

Before returning, compare the interpretation against result summaries, fit or
inference summaries, residuals, count windows, significance proxies, status
fields, and validation checks. If evidence is weak, unstable, missing, or based
on a limited manifest subset, use cautious language.
