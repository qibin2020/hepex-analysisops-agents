# Hyy L1 Strict Skill

L1 is a strict baseline task. Use the leading and subleading photons at indices
0 and 1, apply the exact baseline cuts from the prompt, build `m_yy` directly
from the provided GeV-scale branch values, histogram 100-160 GeV in 1 GeV bins,
assign sqrt(N) uncertainties, and fit Gaussian signal plus polynomial order 4
background over 100-160 GeV.

The trace must include input file provenance, selected event count, cutflow
summary, the exact seven workflow stages, baseline cuts, object definition,
derived observables, histogram definition, fit model family, generated output
files, and a reported signal peak consistent with the fit summary.
