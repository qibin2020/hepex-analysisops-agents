# Hyy L2/L3 Strategy Skill

L2 and L3 allow scientific autonomy, but the result must still be auditable.
Record the visible data scope, selected strategy, photon selection and pairing,
observable construction, spectrum construction, inference/background method,
validation actions, and limitations.

Use cautious interpretation language. A localized excess near 125 GeV can be
reported only when the fit or residual evidence supports it. Do not claim a
discovery from a limited manifest subset or from an unstable fit.
